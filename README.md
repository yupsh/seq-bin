# yup-seq

```
NAME:
   seq - print a sequence of numbers

USAGE:
   seq [OPTIONS] LAST
      seq [OPTIONS] FIRST LAST
      seq [OPTIONS] FIRST INCREMENT LAST

      Print numbers from FIRST to LAST, in steps of INCREMENT.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --separator value, -s value  use STRING to separate numbers (default: 
)
   --format value, -f value     use printf style floating-point FORMAT
   --equal-width, -w            equalize width by padding with leading zeroes (default: false)
   --help, -h                   show help
```
